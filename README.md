# manobot # ManoBot

ManoBot is a Windows utility for optimizing Fortnite performance and input responsiveness.

## Website
https://<Manotweaks>.github.io/manobot/

## Safety
ManoBot modifies Fortnite configuration files and Windows system settings only.
It does not inject code, modify game memory, or interact with anti-cheat systems.
